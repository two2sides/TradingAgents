# 数据源与分析师清单（当前默认安装）

> 原则：**每条数据路径只保留一个正在使用的来源**；其它旧实现仍可能留在代码里，但默认配置不会去调。  
> 更新时间：与 `default_config.data_vendors` / CLI 分析师勾选文案保持一致。

---

## 1. 外部数据：三类

### A. 已有替代方案 · 默认可用 · **你不用配 Key**

| 用途 | 默认唯一来源 | 说明 |
|------|--------------|------|
| 日线行情 OHLCV | `yahoo_chart` | 直连 Chart JSON，**不是**会挂的 `yfinance` 库 |
| 技术指标 | `stockstats` | 本地计算，吃上面的 OHLCV |
| 公司/全球新闻 | `google_news` | Google News RSS |
| 美股基本面 | `sec_edgar` | SEC EDGAR companyfacts（仅美股） |
| Market 扩展工具用价 | 同上 `load_ohlcv` → chart | 多周期 / 量能 / 缺口等分析师工具 |

### B. 可选增强 · **配 Key 才有意义**（默认分析师已不强制调用）

| 用途 | 来源 | 环境变量 | 现状 |
|------|------|----------|------|
| 宏观利率/通胀等 | FRED | `FRED_API_KEY` | 默认 **News Analyst 已不再绑定**；配了 Key 以后可再挂回 |
| LLM 对话 | DeepSeek / OpenAI / … | 对应 `*_API_KEY` | **必须**：否则整图无法跑 |
| Alpha Vantage（备用） | AV | `ALPHA_VANTAGE_API_KEY` | **默认不用**；仅当你想换源时再改 config |

### C. 已放弃进默认路径 · **不必再折腾**

| 旧来源 | 原因 | 处理 |
|--------|------|------|
| `yfinance` Python 包 | 你这边稳定失败（429/crumb） | 默认不调用 |
| Stooq | 常返回 HTML，不可用 | 默认不调用 |
| StockTwits | 403 | Sentiment 默认路径已去掉 |
| Reddit RSS | 429 | Sentiment 默认路径已去掉 |
| 内部人交易 | 无免 Key 稳源 | 默认不提供该工具 |
| Yahoo quoteSummary 基本面 | 401 | 用 SEC 代替 |

## 2. Analysts Team：能否勾选 / 数据是否就绪

CLI 勾选界面已标注数据来源。四个成员：

| CLI 成员 | 可选？ | 依赖的数据路径 | 实现状态 |
|----------|--------|----------------|----------|
| **Market Analyst** | ✅ 可选 | `get_stock_data`→yahoo_chart；`get_indicators`→stockstats；验证快照；多周期 OHLCV tools | **已实现，无需行情 Key** |
| **Sentiment Analyst** | ✅ 可选 | 预取 `get_news`→google_news + 词典打分 | **已实现，无需 Key** |
| **News Analyst** | ✅ 可选 | 强制预取 `get_news` + `get_global_news`→google_news，随后允许 ReAct 补充 | **已实现，无需 Key**；组合 ticker/公司别名/业绩/监管等查询并去重 |
| **Fundamentals Analyst** | ✅ 美股可选；加密资产 CLI 会隐藏 | `sec_edgar` 四表 | **美股已实现，无需 Key**；非美股会空 |

下游固定团队（不可在 Analysts 步取消，跑完分析师后自动进）：Bull / Bear / Research Manager / Trader / Risk 三人 / **Portfolio Manager（终局权威）**。他们主要吃上游报告 + LLM，不再另拉一套行情 Key。详细 Agent↔工具对应见同目录 `扩展方案.md` §3。

## 3. 默认配置一览（唯一生效源）

```text
core_stock_apis      = yahoo_chart
technical_indicators = stockstats
fundamental_data     = sec_edgar
news_data            = google_news
```

文件：`tradingagents/default_config.py`  
CLI 勾选提示：`cli/utils.py` → `ANALYST_ORDER`  
News / Sentiment 工具列表：与 ToolNode 已对齐（只挂新闻相关）。

---

## 4. 你需要做的最少配置

1. **必配**：你选用的 LLM Key（如 `DEEPSEEK_API_KEY`）写在 `.env`。  
2. **不用配**：Yahoo / Stooq / Google News / SEC（默认路径）。  
3. **可选**：`FRED_API_KEY`——只有你以后想给 News 加宏观工具时才需要。
