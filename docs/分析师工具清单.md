# 分析师工具清单（C 侧）

与同目录 `扩展方案.md` §3 同步。  
**[C 新增]** = 本仓库 C 在开发中加入；其余为既有框架能力。

## Market Analyst

| 工具 | 来源 | 作用（白话） | 调用方式 |
|------|------|--------------|----------|
| `get_stock_data` | 既有 | 拉截止分析日的日线 OHLCV | LLM **按需** ReAct |
| `get_indicators` | 既有 | RSI/MACD/均线等 | LLM **按需** ReAct |
| `get_verified_market_snapshot` | 既有 | 校验用价量快照 | LLM **按需**（提示词要求写报告前调用） |
| `analyze_multi_horizon_ohlcv_for_ticker` | **[C 新增]** | 多窗口涨跌/波动/相对均线 | LLM **按需**（提示词要求至少一次） |
| `detect_volume_anomaly_for_ticker` | **[C 新增]** | 量能是否异常 | 同上 |
| `detect_price_gap_for_ticker` | **[C 新增]** | 跳空与回补 | 同上 |

## Sentiment Analyst

| 能力 | 来源 | 作用（白话） | 调用方式 |
|------|------|--------------|----------|
| `get_news`（预取） | 既有 | 近 7 日新闻原文 | **代码强制**预取，再塞进 prompt |
| `score_documents`（词典） | **[C 新增]** | 可复现情绪分 | **代码强制**，不是 LLM 自选 Tool |

## News Analyst

| 工具 | 来源 | 调用方式 |
|------|------|----------|
| `get_news` / `get_global_news` | 既有 | **代码强制预取**公司与全球新闻；LLM 可按需 ReAct 补充 |

Google News 无 Key 路径会组合 ticker、常见公司别名、业绩/指引/监管/产品等查询并
按标题与来源去重；报告必须披露新闻稀疏，不能用模型记忆补造标题。

## Bull / Bear Researcher（本轮精简）

| 工具 | 作用（白话） | 调用方式 |
|------|--------------|----------|
| `compare_to_benchmark_for_ticker` | 相对 SPY/区域指数 5/20/60 日 Alpha | LLM 按需；**一次完整 run 最多执行一次** |
| `get_drawdown_risk_stats_for_ticker` | 回撤、距高点天数、自低点反弹 | LLM 按需；**一次完整 run 最多执行一次** |

> Bull/Bear 仍完整阅读四份分析师报告 + 辩论历史；工具用于**补强可核对数字**，不是替代阅读。
> 多周期、量能、缺口由 Market Analyst 负责，新闻情绪由 Sentiment Analyst
> 负责；对应重复辩论工具已删除。工具一旦执行会记录在
> `investment_debate_state.used_tools`，后续轮次不再向模型提供。
> 每次调用同时写入 `audit/event_trace.jsonl`；Bull/Bear 每轮另保存
> `DebateTurnRecord`，完整周期没有新增 Artifact 时可提前停止。
